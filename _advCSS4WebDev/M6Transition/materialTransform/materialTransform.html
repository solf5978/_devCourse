5.2.4 Cards

  • HTML notes:
    • Card div = card's physical area
    • Card-inner = what flips
    • Card-front = normal content
    • Card-back = rotateY(180deg)
  • CSS notes:
    • Card = set size + perspective
    • Card-inner = transition + preserve-3d
    • Card-face = backface-visibility: hidden
    • Card-back = rotateY(180deg)
    • Card-inner:hover = rotateY(180deg)
↑ ⇐Top

5.2.5 Carousels

  • HTML notes:
    • Carousel-scene = container
    • Carousel = what we will rotate
    • Cells = multiple images / slides
  • CSS notes:
    • Carousel-scene = setup, set perspective
    • Carousel = preserve-3d, transition
    • Cells = divide # cells by 360, rotateY, + translateZ
    • Keyframes = rotateY - carousel

5.2.6 Cubes & Boxes

  • HTML notes:
    • Cube-scene = container
    • Cube = what we will rotate
    • Cube-face = multiple images
  • CSS notes:
    • Cube-scene = width, height, perspective
    • Cube = preserve-3d, rotate3d, transition
    • Cube:hover = rotate3d differently
    • Cube-face = Size, rotateY, translateZ (1/2 size)
